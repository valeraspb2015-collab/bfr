# БФР - Best Flat Rent

Платформа для аренды квартир в России, соединяющая гостей напрямую с владельцами недвижимости.

## Архитектура

- **Frontend**: React + TypeScript + Vite + Wouter (роутинг)
- **Backend**: Express + TypeScript
- **Хранилище**: PostgreSQL (DbStorage через пакет `pg`) — данные сохраняются постоянно
- **UI**: Shadcn/ui + Tailwind CSS (с дизайн-токенами)
- **Мессенджеры**: Telegram (@bfrreplit_bot) + WhatsApp

## Основные функции

### Для гостей
- Форма подбора квартиры (`/request`)
- Поля: имя, телефон, район, бюджет, комнаты, даты заезда/выезда
- Отправка заявки в Telegram + WhatsApp

### Для хозяев
- Форма заявки на присоединение (`/owner-application`)
- Поля: город, имя, телефон, ссылка на квартиру, вопрос
- Отправка в Telegram + WhatsApp

### Админ-панель (`/admin`)
- Просмотр всех заявок от гостей
- Фильтрация по статусу (новая, в работе, завершена)
- **Ответ гостям на их номера телефонов** через:
  - WhatsApp (с предзаполненным сообщением)
  - Telegram (прямая ссылка на номер)
  - Телефонный звонок (tel: ссылка)

## Интеграция с мессенджерами

### Telegram бот
- Бот: @bfrreplit_bot
- Токен: TELEGRAM_BOT_TOKEN (secret)
- Отправка уведомлений о новых заявках (опционально)

### WhatsApp
- Основной метод связи с гостями и хозяевами
- Используется для:
  - Уведомлений (гости: +79899865887, хозяева: +79213798941)
  - Ответов гостям через админ-панель

### Примечание об SMS
Пользователь отклонил интеграцию Twilio для отправки SMS. Вместо этого используются:
- WhatsApp direct links (предпочтительный метод)
- Telegram direct links
- Телефонные звонки через tel: links

Все методы связи доступны в админ-панели по адресу `/admin`.

## Дизайн

- PayPal-inspired двухколоночный лендинг
- Диагональный разделитель между секциями (clipPath)
- Эмоциональные изображения:
  - Гости: семья с багажом на вокзале (AI-generated)
  - Хозяева: landlord с ключами (stock image)

## Контакты (спрятаны от пользователей)

Номера телефонов находятся только в серверной части (`server/routes.ts`) и не отображаются на сайте для безопасности.

## Структура проекта

```
client/
  src/
    components/
      - AdminPanel.tsx (панель администратора с возможностью ответа гостям)
      - Hero.tsx (главная секция)
      - FAQ.tsx (часто задаваемые вопросы с Accordion)
      - HowItWorks.tsx (инструкция)
      - RequestForm.tsx (форма для гостей)
      - OwnerApplicationForm.tsx (форма для хозяев)
      - Footer.tsx
    pages/
      - landing.tsx
      - request.tsx
      - owner-application.tsx
      - admin.tsx
server/
  - routes.ts (API endpoints + WhatsApp/Telegram integration)
  - storage.ts (in-memory data storage)
shared/
  - schema.ts (типы и схемы данных)
```
